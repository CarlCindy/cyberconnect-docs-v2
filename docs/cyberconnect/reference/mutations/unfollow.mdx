`unfollow` unfollow an address. This mutation requires an API key in a `X-API-KEY` header.

```graphql
unfollow(
  input: UnfollowInput!
): UnfollowResponse!
```


### Arguments

#### `input` ([`UnfollowInput`](/cyberconnect/reference/inputs/unfollow-input))




### Type

#### [`UnfollowResponse`](/cyberconnect/reference/objects/unfollow-response) 

The `UnfollowResponse` type returns `unfollow` request status.
