---


`profileByID` provides information about a profile based on the profile's `profileID` input.

```graphql
profileByID(
  profileID: ProfileID!
): Profile
```


### Arguments

#### `profileID` ([`ProfileID`](/cyberconnect/reference/scalars/profile-id))




### Type

#### [`Profile`](/cyberconnect/reference/objects/profile) 

The `Profile` type provides information about a profile.
