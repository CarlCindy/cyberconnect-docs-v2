---


`profilesByHandles` provides information about the profiles based on the profiles' `handles` list input.

```graphql
profilesByHandles(
  chainID: ChainID
  handles: [String!]!
): [Profile]
```


### Arguments

#### `chainID` ([`ChainID`](/cyberconnect/reference/scalars/chain-id))




#### `handles` ([`String`](/cyberconnect/reference/scalars/string))




### Type

#### [`Profile`](/cyberconnect/reference/objects/profile) 

The `Profile` type provides information about a profile.
