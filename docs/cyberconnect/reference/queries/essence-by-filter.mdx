`essencesByFilter` returns an essence with corresponding filter fields

```graphql
essenceByFilter(
  appID: String
  lang: String
  tags: [String!]
): [Essence]
```

#### `lang` ([`String`](/reference/scalars/string))




#### `tags` ([`String`](/reference/scalars/string))
