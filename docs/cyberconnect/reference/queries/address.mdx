`address` returns the information about an EVM address

```graphql
address(
  address: AddressEVM!
): Address
```


### Arguments

#### `address` ([`AddressEVM`](/cyberconnect/reference/scalars/address-evm))




### Type

#### [`Address`](/cyberconnect/reference/objects/address) 

The `Address` type provides an address information on multiple EVM chains.
