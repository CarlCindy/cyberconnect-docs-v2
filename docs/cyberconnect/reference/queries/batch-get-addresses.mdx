`batchGetAddresses` returns information about a list of EVM addresses on a certain chain.

```graphql
batchGetAddresses(
  addresses: [AddressEVM!]!
  chainID: ChainID!
): [Address!]!
```


### Arguments

#### `addresses` ([`AddressEVM`](/reference/scalars/address-evm))




#### `chainID` ([`ChainID`](/reference/scalars/chain-id))




### Type

#### [`Address`](/reference/objects/address) 

The `Address` type provides an address information on multiple EVM chains.
