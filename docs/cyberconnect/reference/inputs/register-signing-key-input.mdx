---


The `RegisterSigningKeyInput` input type specifies the params required to register a signing key.

```graphql
input RegisterSigningKeyInput {
  address: String!
  pubKey: String!
}
```


### Fields

#### `address` ([`String`](/reference/scalars/string))

`address` the user's address.


#### `pubKey` ([`String`](/reference/scalars/string))

`pubKey` the generated public key.
