The `CreateAPIKeyInput` input type specifies the params required to create an `API key`.

```graphql
input CreateAPIKeyInput {
  email: String!
  twitterID: String!
  origin: String!
}
```


### Fields

#### `email` ([`String`](/reference/scalars/string))

`email` the user's email address.


#### `twitterID` ([`String`](/reference/scalars/string))

`twitterID` the user's twitter ID.


#### `origin` ([`String`](/reference/scalars/string))

`origin` the domain/host.
