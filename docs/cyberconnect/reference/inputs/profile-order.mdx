The `ProfileOrder` input type specifies the order in which results are returned.

```graphql
input ProfileOrder {
  direction: OrderDirection!
  field: ProfileOrderField
}
```


### Fields

#### `direction` ([`OrderDirection`](/cyberconnect/reference/enums/order-direction))

`direction` specifies the order direction.


#### `field` ([`ProfileOrderField`](/cyberconnect/reference/enums/profile-order-field))

`field` specifies the order field.
