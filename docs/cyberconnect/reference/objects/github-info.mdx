The `GithubInfo` type provides information about a github profile.

```graphql
type GithubInfo {
  username: String!
  gistID: String!
  userID: Int!
}
```


### Fields

#### `username` ([`String`](/reference/scalars/string))

`username` the github profile username.


#### `gistID` ([`String`](/reference/scalars/string))

`gistID` the gist ID used for verification.


#### `userID` ([`Int`](/reference/scalars/int))

`userID` the github profile user ID.
