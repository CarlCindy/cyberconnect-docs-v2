The `FollowStats` type provides stats about a user's followers and followings.

```graphql
type FollowStats {
  followerCount: Int!
  followingCount: Int!
}
```


### Fields

#### `followerCount` ([`Int`](/cyberconnect/reference/scalars/int))

`followerCount` the total number of followers.


#### `followingCount` ([`Int`](/cyberconnect/reference/scalars/int))

`followingCount` the total number of followings.
