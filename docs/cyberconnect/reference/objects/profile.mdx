The `Profile` type provides information about a profile.

```graphql
type Profile implements Node {
  id: ID!
  profileID: Int!
  handle: String!
  owner: Wallet!
  metadata: URL
  metadataInfo: MetadataDetail
  avatar: URL
  externalMetadataInfo: ExternalMetadataDetail
  mintingBlock: Int!
  isPrimary: Boolean!
  operators: [AddressEVM!]!
  essences(
  after: Cursor
  first: Int
  before: Cursor
  last: Int
  orderBy: EssenceOrder
): EssenceConnection!
  subscribers(
  after: Cursor
  first: Int
  before: Cursor
  last: Int
  orderBy: SubscribeOrder
): SubscribeConnection!
  subscribeNFT: ContractAddressEVM
  subscribeMw: MiddlewareInfo
  namespace: Namespace!
  subscriberCount: Int!
  isSubscribedByMe(
  me: AddressEVM!
): Boolean!
}
```


### Fields

#### `id` ([`ID`](/reference/scalars/id))

`id` the profile's unique identifier.


#### `profileID` ([`Int`](/reference/scalars/int))

`profileID` the profile's token id.


#### `handle` ([`String`](/reference/scalars/string))

`handle` the profile's handle.


#### `owner` ([`Wallet`](/reference/objects/wallet))

`owner` the profile's owner wallet.


#### `metadata` ([`URL`](/reference/scalars/url))

`metadata` the profile's metadata url.


#### `metadataInfo` ([`MetadataDetail`](/reference/objects/metadata-detail))

`metadataInfo` information about the profile's metadata.


#### `avatar` ([`URL`](/reference/scalars/url))

`avatar` the profile's avatar url.


#### `externalMetadataInfo` ([`ExternalMetadataDetail`](/reference/objects/external-metadata-detail))

`externalMetadataInfo` informantion about the profile's metadata on Link3.


#### `mintingBlock` ([`Int`](/reference/scalars/int))

`mintingBlock` the block number when the profile was minted.


#### `isPrimary` ([`Boolean`](/reference/scalars/boolean))

`isPrimary` whether the profile is the primary profile of the owner.


#### `operators` ([`AddressEVM`](/reference/scalars/address-evm))

`operators` the profile's list of operator addresses


#### `essences` ([`EssenceConnection`](/reference/objects/essence-connection))

`essences` provides a paginated list of ordered essences.
- ##### `after` ([`Cursor`](/reference/scalars/cursor))

`after` the cursor to retreive results after.


- ##### `first` ([`Int`](/reference/scalars/int))

`first` the maximum number of essences to return after a cursor.


- ##### `before` ([`Cursor`](/reference/scalars/cursor))

`before` the cursor to retreive results before.


- ##### `last` ([`Int`](/reference/scalars/int))

`last` the maximum number of essences to return before a cursor.


- ##### `orderBy` ([`EssenceOrder`](/reference/inputs/essence-order))

`orderBy` the order in which results are returned.


#### `subscribers` ([`SubscribeConnection`](/reference/objects/subscribe-connection))

`subscribers` provides a paginated list of ordered subscribers.
- ##### `after` ([`Cursor`](/reference/scalars/cursor))

`after` the cursor to retreive results after.


- ##### `first` ([`Int`](/reference/scalars/int))

`first` the maximum number of subscribers to return after a cursor.


- ##### `before` ([`Cursor`](/reference/scalars/cursor))

`before` the cursor to retreive results before.


- ##### `last` ([`Int`](/reference/scalars/int))

`last` the maximum number of subscribers to return before a cursor.


- ##### `orderBy` ([`SubscribeOrder`](/reference/inputs/subscribe-order))

`orderBy` the order in which results are returned.


#### `subscribeNFT` ([`ContractAddressEVM`](/reference/scalars/contract-address-evm))

`subscribeNFT` the profile's subscribe NFT contract address.


#### `subscribeMw` ([`MiddlewareInfo`](/reference/objects/middleware-info))

`subscribeMw` information about the profile's subscribe middleware.


#### `namespace` ([`Namespace`](/reference/objects/namespace))

`namespace` information about the profile's namespace.


#### `subscriberCount` ([`Int`](/reference/scalars/int))

`subscriberCount` the number of subscribers.


#### `isSubscribedByMe` ([`Boolean`](/reference/scalars/boolean))

`isSubscribedByMe` whether the given address has subscribed to the profile.
- ##### `me` ([`AddressEVM`](/reference/scalars/address-evm))
