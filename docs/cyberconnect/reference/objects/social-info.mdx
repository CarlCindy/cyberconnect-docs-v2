---


The `SocialInfo` type provides information about an identity's social profiles.

```graphql
type SocialInfo {
  twitter: TwitterInfo!
  github: GithubInfo!
}
```


### Fields

#### `twitter` ([`TwitterInfo`](/reference/objects/twitter-info))

`twitter` the twitter profile.


#### `github` ([`GithubInfo`](/reference/objects/github-info))

`github` the github profile.
