The `Twitter` type provides the Twitter account info.

```graphql
type Twitter {
  id: String!
  handle: String!
  name: String!
  avatar: URL!
}
```


### Fields

#### `id` ([`String`](/cyberconnect/reference/scalars/string))

`id` the id of the twitter.


#### `handle` ([`String`](/cyberconnect/reference/scalars/string))

`handle` the handle of the twitter.


#### `name` ([`String`](/cyberconnect/reference/scalars/string))

`name` the display name of the twitter.


#### `avatar` ([`URL`](/cyberconnect/reference/scalars/url))

`avatar` the avatar of the twitter.
