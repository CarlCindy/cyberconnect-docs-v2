The `Essence` type provides information about an essence.

```graphql
type Essence implements Node {
  id: ID!
  essenceID: Int!
  name: String!
  symbol: String!
  tokenURI: URL!
  collectMw: MiddlewareInfo!
  collectedBy(
  after: Cursor
  first: Int
  before: Cursor
  last: Int
  orderBy: CollectOrder
): CollectConnection!
  isCollectedByMe(
  me: AddressEVM!
): Boolean!
  createdBy: Profile!
}
```


### Fields

#### `id` ([`ID`](/cyberconnect/reference/scalars/id))

`id` the unique identifier of the essence.


#### `essenceID` ([`Int`](/cyberconnect/reference/scalars/int))

`essenceID` the essence token id.


#### `name` ([`String`](/cyberconnect/reference/scalars/string))

`name` the name of the essence.


#### `symbol` ([`String`](/cyberconnect/reference/scalars/string))

`symbol` the symbol of the essence.


#### `tokenURI` ([`URL`](/cyberconnect/reference/scalars/url))

`tokenURI` the token URI of the essence.


#### `collectMw` ([`MiddlewareInfo`](/cyberconnect/reference/objects/middleware-info))

`collectMw` information about the essence's collect middleware.


#### `collectedBy` ([`CollectConnection`](/cyberconnect/reference/objects/collect-connection))

`collectedBy` provides a paginated list of profiles that have collected this essence.
- ##### `after` ([`Cursor`](/cyberconnect/reference/scalars/cursor))

`after` the cursor to retreive results after.


- ##### `first` ([`Int`](/cyberconnect/reference/scalars/int))

`first` the maximum number of profiles to return after a cursor.


- ##### `before` ([`Cursor`](/cyberconnect/reference/scalars/cursor))

`before` the cursor to retreive results before.


- ##### `last` ([`Int`](/cyberconnect/reference/scalars/int))

`last` the maximum number of profiles to return before a cursor.


- ##### `orderBy` ([`CollectOrder`](/cyberconnect/reference/inputs/collect-order))

`orderBy` the order in which results are returned.


#### `isCollectedByMe` ([`Boolean`](/cyberconnect/reference/scalars/boolean))

`isCollectedByMe` whether the given address has collected this essence.
- ##### `me` ([`AddressEVM`](/cyberconnect/reference/scalars/address-evm))




#### `createdBy` ([`Profile`](/cyberconnect/reference/objects/profile))

`createdBy` information about the profile that created this essence.


### Interfaces

#### [`Node`](/cyberconnect/reference/interfaces/node) 

The `Node` interface follows relay.dev spec.
