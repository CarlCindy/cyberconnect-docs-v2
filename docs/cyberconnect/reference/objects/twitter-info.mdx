The `TwitterInfo` type provides information about a twitter profile.

```graphql
type TwitterInfo {
  handle: String!
  avatar: String!
  verified: Boolean!
  tweetID: String!
  source: String!
  followerCount: Int!
}
```


### Fields

#### `handle` ([`String`](/reference/scalars/string))

`handle` the twitter profile handle.


#### `avatar` ([`String`](/reference/scalars/string))

`avatar` the twitter profile avatar.


#### `verified` ([`Boolean`](/reference/scalars/boolean))

`verified` the verification status of the twitter profile.


#### `tweetID` ([`String`](/reference/scalars/string))

`tweetID` the ID of the tweet used for verification.


#### `source` ([`String`](/reference/scalars/string))

`source` the data source of twitter handle.


#### `followerCount` ([`Int`](/reference/scalars/int))

`followerCount` the number of followers on twitter.
