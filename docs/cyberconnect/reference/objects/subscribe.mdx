The `Subscribe` type provides information about a subscription.

```graphql
type Subscribe {
  wallet: Wallet!
  profile: Profile!
  blockNumber: Int!
}
```


### Fields

#### `wallet` ([`Wallet`](/cyberconnect/reference/objects/wallet))

`wallet` the wallet address of the Subscriber.


#### `profile` ([`Profile`](/cyberconnect/reference/objects/profile))

`profile` the profile of the Subscriber.


#### `blockNumber` ([`Int`](/cyberconnect/reference/scalars/int))

`blockNumber` the block number at which the subscription was created.
