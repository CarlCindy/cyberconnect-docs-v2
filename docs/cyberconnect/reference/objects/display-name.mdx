The `DisplayName` type provides the profile display name on Link3.

```graphql
type DisplayName {
  value: String!
  type: NameType!
}
```


### Fields

#### `value` ([`String`](/cyberconnect/reference/scalars/string))

`value` the value of the display name.


#### `type` ([`NameType`](/cyberconnect/reference/enums/name-type))

`type` the type of the display name.
