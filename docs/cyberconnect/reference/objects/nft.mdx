The `NFT` type provides the NFT data.

```graphql
type NFT {
  chainId: ChainID!
  contract: ContractAddressEVM!
  tokenId: String!
  type: ERCType!
  name: String!
  image: String!
  owner: AddressEVM!
}
```


### Fields

#### `chainId` ([`ChainID`](/cyberconnect/reference/scalars/chain-id))

`chainId` the chain ID of the NFT.


#### `contract` ([`ContractAddressEVM`](/cyberconnect/reference/scalars/contract-address-evm))

`contract` the contract address of the NFT.


#### `tokenId` ([`String`](/cyberconnect/reference/scalars/string))

`tokenId` the token ID of the NFT.


#### `type` ([`ERCType`](/cyberconnect/reference/enums/erctype))

`type` the ERC type of the NFT.


#### `name` ([`String`](/cyberconnect/reference/scalars/string))

`name` the name of the NFT.


#### `image` ([`String`](/cyberconnect/reference/scalars/string))

`image` the content or url of the NFT image.


#### `owner` ([`AddressEVM`](/cyberconnect/reference/scalars/address-evm))

`owner` the owner's address of the NFT.
