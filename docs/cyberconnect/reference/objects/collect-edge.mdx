The `CollectEdge` type provides a `Collect` node along with its cursor.

```graphql
type CollectEdge {
  node: Collect
  cursor: Cursor!
}
```


### Fields

#### `node` ([`Collect`](/cyberconnect/reference/objects/collect))

`node` the node of the `Collect` object.


#### `cursor` ([`Cursor`](/cyberconnect/reference/scalars/cursor))

`cursor` the cursor of the `Collect` node.
