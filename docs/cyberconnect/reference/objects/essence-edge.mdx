title: EssenceEdge
---


The `EssenceEdge` type provides an `Essence` node along with its cursor.

```graphql
type EssenceEdge {
  node: Essence
  cursor: Cursor!
}
```


### Fields

#### `node` ([`Essence`](/reference/objects/essence))

`node` the node of the `Essence` object.


#### `cursor` ([`Cursor`](/reference/scalars/cursor))

`cursor` the cursor of the `Essence` node.
