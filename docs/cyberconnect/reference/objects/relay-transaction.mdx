The `RelayTransaction` type contains the information about a relayed transaction.

```graphql
type RelayTransaction implements Node {
  id: ID!
  txHash: String!
  typedData: TypedData
}
```


### Fields

#### `id` ([`ID`](/reference/scalars/id))

`id` the relayed transaction ID.


#### `txHash` ([`String`](/reference/scalars/string))

`txHash` the transaction hash.


#### `typedData` ([`TypedData`](/reference/objects/typed-data))

`typedData` information about the Typed Data.


### Interfaces

#### [`Node`](/reference/interfaces/node) 

The `Node` interface follows relay.dev spec.
