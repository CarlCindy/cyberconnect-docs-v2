The `LoginGetMessageInput` input type specifies the Typed Data message parameters.

```graphql
input LoginGetMessageInput {
  domain: String!
  address: AddressEVM!
}
```


### Fields

#### `domain` ([`String`](/cyberconnect/reference/scalars/string))

`domain` the domain of the app.


#### `address` ([`AddressEVM`](/cyberconnect/reference/scalars/address-evm))

`address` the address of the user.
