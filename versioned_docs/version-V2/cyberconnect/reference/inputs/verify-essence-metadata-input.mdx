No description

```graphql
input VerifyEssenceMetadataInput {
  version: String!
  name: String
  app_id: String
  description: String
  image: URL
  issue_date: String
  external_url: URL
  lang: String
  tags: [String!]
  media: [Media!]
  attributes: [Attribute!]
  image_data: String
}
```


### Fields

#### `version` ([`String`](/cyberconnect/reference/scalars/string))

`version` Version of the metadata schema used for the issued item. Only support `1.0.0` for now.


#### `name` ([`String`](/cyberconnect/reference/scalars/string))

`name` Name of the item.


#### `app_id` ([`String`](/cyberconnect/reference/scalars/string))

`app_id` Id of the application under which the items are being minted.


#### `description` ([`String`](/cyberconnect/reference/scalars/string))

`description` Description of the item.


#### `image` ([`URL`](/cyberconnect/reference/scalars/url))

`image` URL to the image of the item.


#### `issue_date` ([`String`](/cyberconnect/reference/scalars/string))

`issue_date` Creation time of the item as ISO 8601.


#### `external_url` ([`URL`](/cyberconnect/reference/scalars/url))

`external_url` URL to the item on your site.


#### `lang` ([`String`](/cyberconnect/reference/scalars/string))

`lang` Language of the content as a BCP47 language tag.


#### `tags` ([`String`](/cyberconnect/reference/scalars/string))

`tags` Field indicating the tags associated with the content. Limited to max. 5 tags.


#### `media` ([`Media`](/cyberconnect/reference/inputs/media))

`media` Media refers to any image, video, or any other MIME type attached to the content. Limited to max. 10 media objects.


#### `attributes` ([`Attribute`](/cyberconnect/reference/inputs/attribute))

`attributes` Attributes for the item.


#### `image_data` ([`String`](/cyberconnect/reference/scalars/string))

`image_data` SVG image data when the image is not passed. Only use this if you're not including the image parameter.
