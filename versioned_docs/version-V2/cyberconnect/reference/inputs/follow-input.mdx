The `FollowInput` input type specifies the params required to follow an address.

```graphql
input FollowInput {
  fromAddr: String!
  toAddr: String!
  signature: String!
  operation: String!
  signingKey: String!
}
```


### Fields

#### `fromAddr` ([`String`](/cyberconnect/reference/scalars/string))

`fromAddr` the user's address.


#### `toAddr` ([`String`](/cyberconnect/reference/scalars/string))

`toAddr` the address the user wants to follow.


#### `signature` ([`String`](/cyberconnect/reference/scalars/string))

`signature` the signature from signing the `operation` json string.


#### `operation` ([`String`](/cyberconnect/reference/scalars/string))

`operation` the operation json string to sign.


#### `signingKey` ([`String`](/cyberconnect/reference/scalars/string))

`signingKey` the registered signing key.
