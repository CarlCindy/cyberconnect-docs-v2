The `UnfollowInput` input type specifies the params required to unfollow an address.

```graphql
input UnfollowInput {
  fromAddr: String!
  toAddr: String!
  signature: String!
  operation: String!
  signingKey: String!
}
```


### Fields

#### `fromAddr` ([`String`](/reference/scalars/string))

`fromAddr` the user's address.


#### `toAddr` ([`String`](/reference/scalars/string))

`toAddr` the address the user wants to unfollow.


#### `signature` ([`String`](/reference/scalars/string))

`signature` the signature from signing the `operation` json string.


#### `operation` ([`String`](/reference/scalars/string))

`operation` the operation json string to sign.


#### `signingKey` ([`String`](/reference/scalars/string))

`signingKey` the registered signing key.
