`registerSigningKey` registers a signing key to use with `follow` and `unfollow` inputs.

```graphql
registerSigningKey(
  input: RegisterSigningKeyRequest!
): RegisterSigningKeyResponse!
```


### Arguments

#### `input` ([`RegisterSigningKeyRequest`](/cyberconnect/reference/inputs/register-signing-key-request))




### Type

#### [`RegisterSigningKeyResponse`](/cyberconnect/reference/objects/register-signing-key-response) 

The `RegisterKeyResponse` type returns `registerSigningKey` request status.
