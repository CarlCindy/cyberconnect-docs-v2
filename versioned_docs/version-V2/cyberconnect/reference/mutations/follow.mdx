`follow` follow an address. This mutation requires an API key in a `X-API-KEY` header.

```graphql
follow(
  input: FollowInput!
): FollowResponse!
```


### Arguments

#### `input` ([`FollowInput`](/cyberconnect/reference/inputs/follow-input))




### Type

#### [`FollowResponse`](/cyberconnect/reference/objects/follow-response) 

The `FollowResponse` type returns `follow` request status.
