`nodes` returns a list of nodes based on a set of `node IDs`.

```graphql
nodes(
  ids: [ID!]!
): [Node]!
```


### Arguments

#### `ids` ([`ID`](/reference/scalars/id))




### Type

#### [`Node`](/reference/interfaces/node) 

The `Node` interface follows relay.dev spec.
