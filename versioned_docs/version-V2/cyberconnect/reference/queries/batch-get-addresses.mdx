`batchGetAddresses` returns information about a list of EVM addresses.

```graphql
batchGetAddresses(
  addresses: [AddressEVM!]!
): [Address!]!
```


### Arguments

#### `addresses` ([`AddressEVM`](/cyberconnect/reference/scalars/address-evm))




### Type

#### [`Address`](/cyberconnect/reference/objects/address) 

The `Address` type provides an address information on multiple EVM chains.
