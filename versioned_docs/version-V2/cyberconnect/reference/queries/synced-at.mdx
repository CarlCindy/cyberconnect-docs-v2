title: syncedAt
---


`syncedAt` returns the block information about the latest block when the `chainID` was last synced.

```graphql
syncedAt(
  chainID: ChainID
): BlockchainTime!
```


### Arguments

#### `chainID` ([`ChainID`](/reference/scalars/chain-id))




### Type

#### [`BlockchainTime`](/reference/objects/blockchain-time) 

The `BlockchainTime` type provides block information.
