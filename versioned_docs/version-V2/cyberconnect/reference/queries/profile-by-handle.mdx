`profileByHandle` provides information about a profile based on the profile's `handle` input.

```graphql
profileByHandle(
  handle: String!
): Profile
```


### Arguments

#### `handle` ([`String`](/cyberconnect/reference/scalars/string))




### Type

#### [`Profile`](/cyberconnect/reference/objects/profile) 

The `Profile` type provides information about a profile.
