The `Headline` type provides the data in the headline of Link3 personal profiles.

```graphql
type Headline {
  twitter: Twitter!
  title: String!
}
```


### Fields

#### `twitter` ([`Twitter`](/cyberconnect/reference/objects/twitter))

`twitter` the Twitter account info in headline.


#### `title` ([`String`](/cyberconnect/reference/scalars/string))

`title` the title of the headline.
