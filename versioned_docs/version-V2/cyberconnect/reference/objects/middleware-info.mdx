The `MiddlewareInfo` type provides information about a middleware.

```graphql
type MiddlewareInfo {
  contractAddress: ContractAddressEVM!
  type: MiddlewareType!
  data: String!
}
```


### Fields

#### `contractAddress` ([`ContractAddressEVM`](/reference/scalars/contract-address-evm))

`contractAddress` the middleware contract address.


#### `type` ([`MiddlewareType`](/reference/enums/middleware-type))

`type` the middleware type.


#### `data` ([`String`](/reference/scalars/string))

`data` the middleware data.
