The `Follow` type provides information about a follow relationship.

```graphql
type Follow {
  from: Address!
  to: Address!
}
```


### Fields

#### `from` ([`Address`](/reference/objects/address))

`from` the source address.


#### `to` ([`Address`](/reference/objects/address))

`to` the address followed by `from` address.
