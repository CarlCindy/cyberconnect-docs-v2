The `FollowEdge` type provides a `Follow` node along with its cursor.

```graphql
type FollowEdge {
  node: FollowStatus
  cursor: Cursor!
}
```


### Fields

#### `node` ([`FollowStatus`](/cyberconnect/reference/objects/follow-status))

`node` the node of the `Follow` object.


#### `cursor` ([`Cursor`](/cyberconnect/reference/scalars/cursor))

`cursor` the cursor of the `Follow` node.
