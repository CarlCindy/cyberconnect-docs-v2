The `Mention` type provides the data in the mention sections of Link3 profiles.

```graphql
type Mention {
  twitter: Twitter!
  title: String!
  description: String!
}
```


### Fields

#### `twitter` ([`Twitter`](/cyberconnect/reference/objects/twitter))

`twitter` the Twitter account info in the mention section.


#### `title` ([`String`](/cyberconnect/reference/scalars/string))

`title` the title of the twitter.


#### `description` ([`String`](/cyberconnect/reference/scalars/string))

`description` the description of the twitter.
