The `FollowStatus` type provides information about a follow relationship.

```graphql
type FollowStatus {
  address: AddressEVM!
  handle: String!
  proof: Proof!
}
```


### Fields

#### `address` ([`AddressEVM`](/cyberconnect/reference/scalars/address-evm))

`address` the source address.


#### `handle` ([`String`](/cyberconnect/reference/scalars/string))

`handle` the profile handle followed by `address`.


#### `proof` ([`Proof`](/cyberconnect/reference/objects/proof))

`proof` the proof of the off-chain follow operation.
