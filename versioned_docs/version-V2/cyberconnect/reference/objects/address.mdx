The `Address` type provides an address information on multiple EVM chains.

```graphql
type Address {
  address: AddressEVM!
  chainID: ChainID!
  wallet: Wallet
  followings(
  offset: Int
  limit: Int
): FollowConnection!
  followingCount: Int!
}
```


### Fields

#### `address` ([`AddressEVM`](/cyberconnect/reference/scalars/address-evm))

`address` the address.


#### `chainID` ([`ChainID`](/cyberconnect/reference/scalars/chain-id))

`chainID` the ChainID.


#### `wallet` ([`Wallet`](/cyberconnect/reference/objects/wallet))

`wallet` the wallet address on Ethereum.


#### `followings` ([`FollowConnection`](/cyberconnect/reference/objects/follow-connection))

`followings` provides a paginated list of ordered followings profiles.
- ##### `offset` ([`Int`](/cyberconnect/reference/scalars/int))




- ##### `limit` ([`Int`](/cyberconnect/reference/scalars/int))




#### `followingCount` ([`Int`](/cyberconnect/reference/scalars/int))

`followingCount` returns stats about a user's followers and followings
