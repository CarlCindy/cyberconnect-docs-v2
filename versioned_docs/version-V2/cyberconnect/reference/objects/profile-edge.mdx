The `SubscribeEdge` type provides a `Profile` node along with its cursor.

```graphql
type ProfileEdge {
  node: Profile
  cursor: Cursor!
}
```


### Fields

#### `node` ([`Profile`](/cyberconnect/reference/objects/profile))

`node` the node of the `Profile` object.


#### `cursor` ([`Cursor`](/cyberconnect/reference/scalars/cursor))

`cursor` the cursor of the `Profile` node.
