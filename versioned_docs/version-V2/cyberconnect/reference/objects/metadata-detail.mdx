The `MetadataDetail` type provides information about a profile's metadata.

```graphql
type MetadataDetail {
  name: String!
  bio: String!
  type: ProfileType!
  title: String!
  organization: String!
  nameType: NameType!
  avatarType: AvatarType!
  version: String!
}
```


### Fields

#### `name` ([`String`](/reference/scalars/string))

`name` the name of the profile.


#### `bio` ([`String`](/reference/scalars/string))

`bio` the bio of the profile.


#### `type` ([`ProfileType`](/reference/enums/profile-type))

`type` the type of the profile.


#### `title` ([`String`](/reference/scalars/string))

`title` the title of the profile.


#### `organization` ([`String`](/reference/scalars/string))

`organisation` the organization of the profile.


#### `nameType` ([`NameType`](/reference/enums/name-type))

`nameType` the name type of the profile.


#### `avatarType` ([`AvatarType`](/reference/enums/avatar-type))

`avatarType` the avatar type of the profile.


#### `version` ([`String`](/reference/scalars/string))

`version` the version of the profile.
