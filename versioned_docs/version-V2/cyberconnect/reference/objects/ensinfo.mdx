The `ENSInfo` type provides information about an ENS domain.

```graphql
type ENSInfo {
  domain: String!
  avatar: String!
}
```


### Fields

#### `domain` ([`String`](/cyberconnect/reference/scalars/string))

`domain` the ENS domain name.


#### `avatar` ([`String`](/cyberconnect/reference/scalars/string))

`avatar` the ENS avatar.
