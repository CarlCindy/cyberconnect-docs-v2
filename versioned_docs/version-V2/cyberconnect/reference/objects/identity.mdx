The `Identity` type provides information about a user's identity.

```graphql
type Identity {
  ens: ENSInfo!
  joinTime: String!
  socialInfo: SocialInfo!
}
```


### Fields

#### `ens` ([`ENSInfo`](/cyberconnect/reference/objects/ensinfo))

`ens` information about the ENS domain.


#### `joinTime` ([`String`](/cyberconnect/reference/scalars/string))

`joinTime` the time of user's first sent transaction.


#### `socialInfo` ([`SocialInfo`](/cyberconnect/reference/objects/social-info))

`socialInfo` information about the social profiles.
