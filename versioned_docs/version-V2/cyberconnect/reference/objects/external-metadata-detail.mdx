The `MetadataDetail` type provides information about a profile's metadata on Link3.

```graphql
type ExternalMetadataDetail {
  type: ProfileType!
  handle: String!
  displayName: DisplayName!
  avatar: Picture!
  background: Picture!
  bio: String!
  organization: Organization
  personal: Personal
  section: [Section!]
}
```


### Fields

#### `type` ([`ProfileType`](/reference/enums/profile-type))

`type` the type of the profile.


#### `handle` ([`String`](/reference/scalars/string))

`handle` the handle of the profile.


#### `displayName` ([`DisplayName`](/reference/objects/display-name))

`displayName` the display name of the profile on Link3.


#### `avatar` ([`Picture`](/reference/objects/picture))

`avatar` the avatar of the profile on Link3.


#### `background` ([`Picture`](/reference/objects/picture))

`background` the background of the profile on Link3.


#### `bio` ([`String`](/reference/scalars/string))

`bio` the bio of the profile on Link3.


#### `organization` ([`Organization`](/reference/objects/organization))

`organization` the exclusive information about the organization profile on Link3. Take this field if `type` = `ORGANIZATION`.


#### `personal` ([`Personal`](/reference/objects/personal))

`personal` the exclusive information about the personal profile on Link3. Take this field if `type` = `PERSONAL`.


#### `section` ([`Section`](/reference/objects/section))

`section` the sections of the profile on Link3.
