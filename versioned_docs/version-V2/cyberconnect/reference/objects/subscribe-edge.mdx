---


The `SubscribeEdge` type provides a `Subscribe` node along with its cursor.

```graphql
type SubscribeEdge {
  node: Subscribe
  cursor: Cursor!
}
```


### Fields

#### `node` ([`Subscribe`](/cyberconnect/reference/objects/subscribe))

`node` the node of the `Subscribe` object.


#### `cursor` ([`Cursor`](/cyberconnect/reference/scalars/cursor))

`cursor` the cursor of the `Subscribe` node.
