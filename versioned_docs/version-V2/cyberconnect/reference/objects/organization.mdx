The `Organization` type provides the exclusive information about organization profile on Link3.

```graphql
type Organization {
  twitterId: String!
  verified: Boolean!
  followersCount: Int!
  cmcTokenId: String!
  sector: String!
  networks: [String!]!
}
```


### Fields

#### `twitterId` ([`String`](/cyberconnect/reference/scalars/string))

`twitterId` the Twitter account ID the organization is bound to on Link3.


#### `verified` ([`Boolean`](/cyberconnect/reference/scalars/boolean))

`verified` the verification status of the organization on Link3. `True`: verified by Link3; `False`: not verified yet.


#### `followersCount` ([`Int`](/cyberconnect/reference/scalars/int))

`followersCount` the followers count of the organization's profile on Link3.


#### `cmcTokenId` ([`String`](/cyberconnect/reference/scalars/string))

`cmcTokenId` the CoinMarketCap token ID of the token issued by the organization's project.


#### `sector` ([`String`](/cyberconnect/reference/scalars/string))

`sector` the subdivision area of the organization's project.


#### `networks` ([`String`](/cyberconnect/reference/scalars/string))

`networks` the blockchains the organization's project is based on.
