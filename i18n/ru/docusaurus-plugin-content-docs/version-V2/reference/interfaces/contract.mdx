Интерфейс `Contract` предоставляет базовые сведения, которые должны быть доступны для всех EVM контрактов.

```graphql
interface Contract {
  contractAddress: ContractAddressEVM!
  name: String!
  contract_creation_time: String!
  block_number: Int!
  block_hash: String!
  bytecode: String!
}
```


### Поля

#### `contractAddress` ([`ContractAddressEVM`](/reference/scalars/contract-address-evm))
`contractAddress` - адрес контракта.



#### `name` ([`String`](/reference/scalars/string))
`name` - название контракта.



#### `contract_creation_time` ([`String`](/reference/scalars/string))

`contract_creation_time` - дата развертывания контракта.


#### `block_number` ([`Int`](/reference/scalars/int))

`block_number` - номер блока, в котором был развернут контракт.


#### `block_hash` ([`String`](/reference/scalars/string))

`block_hash` - хэш блока, в котором был развернут контракт.


#### `bytecode` ([`String`](/reference/scalars/string))

`bytecode` - байт-код контракта.
