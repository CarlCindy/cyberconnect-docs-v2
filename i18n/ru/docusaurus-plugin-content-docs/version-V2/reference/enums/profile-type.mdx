---


Тип перечисления `ProfileType` предоставляет тип профиля.

```graphql
enum ProfileType {
  PERSONAL
  ORGANIZATION
}
```


### Значения

#### `PERSONAL` 

`PERSONAL` - персональный профиль.


#### `ORGANIZATION` 

`ORGANIZATION` - профиль организации.
