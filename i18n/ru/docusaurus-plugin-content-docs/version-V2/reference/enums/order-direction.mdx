title: OrderDirection
---


Тип перечисления `OrderDirection` предоставляет направления сортировки.

```graphql
enum OrderDirection {
  ASC
  DESC
}
```


### Значение

#### `ASC` 

`ASC` - сортировка в порядке возрастания.


#### `DESC` 

`DESC` - сортировка в порядке убывания.
