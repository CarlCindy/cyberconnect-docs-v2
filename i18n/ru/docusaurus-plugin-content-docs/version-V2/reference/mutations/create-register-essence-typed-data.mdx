`createRegisterEssenceTypedData` возвращает сообщение с типизированными данными спецификации EIP-712 для подписи с помощью кошелька, чтобы зарегистрировать сущность. Для этой мутации требуется заголовок `Authorization` с действительным токеном `Bearer`.

```graphql
createRegisterEssenceTypedData(
  input: CreateRegisterEssenceTypedDataInput!
): CreateRegisterEssenceTypedDataResult!
```

#### `input` ([`CreateRegisterEssenceTypedDataInput`](/reference/inputs/create-register-essence-typed-data-input))




### Тип
