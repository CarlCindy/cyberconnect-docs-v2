`unfollow` отменяет подписку на адрес. Для этой мутации требуется ключ API в заголовке `X-API-KEY`.

```graphql
unfollow(
  input: UnfollowInput!
): UnfollowResponse!
```


### Аргументы

#### `input` ([`UnfollowInput`](/reference/inputs/unfollow-input))




### Тип

#### [`UnfollowResponse`](/reference/objects/unfollow-response) 

Тип ответа `UnfollowResponse`  возвращает статус запроса `unfollow`.
