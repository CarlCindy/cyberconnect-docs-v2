`follow` следует за адресом. Для этой мутации требуется ключ API в заголовке `X-API-KEY`.

```graphql
follow(
  input: FollowInput!
): FollowResponse!
```


### Аргументы

#### `input` ([`FollowInput`](/reference/inputs/follow-input))




### Тип

#### [`FollowResponse`](/reference/objects/follow-response) 

Тип `FollowResponse` возвращает статус запроса `follow`.
