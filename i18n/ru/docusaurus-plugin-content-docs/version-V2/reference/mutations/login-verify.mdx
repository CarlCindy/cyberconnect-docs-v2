`loginVerify` возвращает токен доступа для запросов на мутации, которые требуют аутентификации на основе подписанного сообщения типизированных данных ввода.


```graphql
loginVerify(
  input: LoginVerifyInput!
): LoginVerifyResult!
```


### Аргументы

#### `input` ([`LoginVerifyInput`](/reference/inputs/login-verify-input))




### Тип

#### [`LoginVerifyResult`](/reference/objects/login-verify-result) 

Тип `LoginVerifyResult` предоставляет токен доступа для запросов на мутации, требующих аутентификации.
