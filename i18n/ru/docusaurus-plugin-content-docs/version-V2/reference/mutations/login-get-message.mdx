`loginGetMessage` возвращает сообщение с типизированными данными спецификации EIP-712 для входа в кошелек на основе введенных параметров типизированных данных.

```graphql
loginGetMessage(
  input: LoginGetMessageInput!
): LoginGetMessageResult!
```


### Аргументы

#### `input` ([`LoginGetMessageInput`](/reference/inputs/login-get-message-input))




### Тип

#### [`LoginGetMessageResult`](/reference/objects/login-get-message-result) 

Тип `LoginGetMessageResult` предоставляет сообщение данных, типизированных по спецификации EIP-712, для подписи с помощью кошелька.
