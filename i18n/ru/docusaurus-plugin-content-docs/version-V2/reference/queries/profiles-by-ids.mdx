title: profilesByIDs
---


`profilesByIDs` предоставляет информацию о профилях на основе введенного списка `profileIDs` профилей.

```graphql
profilesByIDs(
  chainID: ChainID
  profileIDs: [ProfileID!]!
): [Profile]
```


### Аргументы

#### `chainID` ([`ChainID`](/reference/scalars/chain-id))




#### `profileIDs` ([`ProfileID`](/reference/scalars/profile-id))




### Тип

#### [`Profile`](/reference/objects/profile) 

Тип `Profile` предоставляет информацию о профиле.
