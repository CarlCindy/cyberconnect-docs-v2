`profileByID` предоставляет информацию о профиле на основе введенного `profileID`.

```graphql
profileByID(
  chainID: ChainID
  profileID: ProfileID!
): Profile
```


### Аргументы

#### `chainID` ([`ChainID`](/reference/scalars/chain-id))




#### `profileID` ([`ProfileID`](/reference/scalars/profile-id))




### Тип

#### [`Profile`](/reference/objects/profile) 

Тип `Profile` предоставляет информацию о профиле.
