`nodes` возвращает список узлов, основанный на наборе `node IDs`.

```graphql
nodes(
  ids: [ID!]!
): [Node]!
```


### Аргументы

#### `ids` ([`ID`](/reference/scalars/id))




### Тип

#### [`Node`](/reference/interfaces/node) 

Интерфейс `Node` соответсвует спецификации relay.dev.
