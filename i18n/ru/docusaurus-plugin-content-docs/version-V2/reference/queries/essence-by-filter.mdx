`essencesByFilter` возвращает сущность с соответствующими полями фильтра.

```graphql
essenceByFilter(
  appID: String
  lang: String
  tags: [String!]
): [Essence]
```


### Аргументы

#### `appID` ([`String`](/reference/scalars/string))




#### `lang` ([`String`](/reference/scalars/string))




#### `tags` ([`String`](/reference/scalars/string))




### Тип

#### [`Essence`](/reference/objects/essence) 

Тип `Essence`  предоставляет информацию о сущности.
