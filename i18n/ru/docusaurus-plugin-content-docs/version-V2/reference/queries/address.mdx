`address` возвращает информацию об EVM адресе в определенной цепочке.

```graphql
address(
  address: AddressEVM!
  chainID: ChainID!
): Address
```


### Аргументы

#### `address` ([`AddressEVM`](/reference/scalars/address-evm))




#### `chainID` ([`ChainID`](/reference/scalars/chain-id))




### Тип

#### [`Address`](/reference/objects/address) 

Тип `Address` предоставляет информацию об адресе в нескольких EVM цепочках.
