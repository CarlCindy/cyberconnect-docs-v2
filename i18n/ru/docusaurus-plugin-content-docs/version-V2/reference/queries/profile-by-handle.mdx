`profileByHandle` предоставляет информацию о профиле на основе ввода `handle` профиля.

```graphql
profileByHandle(
  chainID: ChainID
  handle: String!
): Profile
```


### Аргументы

#### `chainID` ([`ChainID`](/reference/scalars/chain-id))




#### `handle` ([`String`](/reference/scalars/string))




### Тип

#### [`Profile`](/reference/objects/profile) 

Тип `Profile` предоставляет информацию о профиле.
