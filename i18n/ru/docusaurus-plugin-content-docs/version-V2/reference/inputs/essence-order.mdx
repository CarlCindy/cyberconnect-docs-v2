---


Тип ввода `EssenceOrder` определяет порядок, в котором возвращаются результаты.

```graphql
input EssenceOrder {
  direction: OrderDirection!
}
```


### Поля

#### `direction` ([`OrderDirection`](/reference/enums/order-direction))

`direction` - указывает направление вывода.
