---

Тип ввода `RelayInput` определяет идентификатор типизированных данных вместе с подписью типизированных данных.

```graphql
input RelayInput {
  typedDataID: ID!
  signature: String!
}
```


### Поля

#### `typedDataID` ([`ID`](/reference/scalars/id))

`typedDataID` - идентификатор типизированных данных.


#### `signature` ([`String`](/reference/scalars/string))

`signature` - подпись сообщения типизированных данных.
