Тип ввода `collectPermissionMwParams` определяет параметры промежуточной программы `collectPermission`.

```graphql
input collectPermissionMwParams {
  signer: AddressEVM!
}
```


### Поля

#### `signer` ([`AddressEVM`](/reference/scalars/address-evm))

`signer` - адрес подписанта.
