---


Тип ввода `CollectEssenceTypedDataOptions` определяет параметры типизированных данных, необходимые для создания сообщения типизированных данных.

```graphql
input CollectEssenceTypedDataOptions {
  overrideNonce: Int
  chainID: Int!
}
```


### Поля

#### `overrideNonce` ([`Int`](/reference/scalars/int))

`overrideNonce` - следует ли переопределять одноразовый номер.


#### `chainID` ([`Int`](/reference/scalars/int))

`chainID` - идентификатор цепочки в сети.
