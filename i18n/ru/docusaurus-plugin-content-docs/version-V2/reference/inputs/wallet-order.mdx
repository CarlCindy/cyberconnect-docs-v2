---


Тип ввода `WalletOrder` определяет порядок, в котором возвращаются результаты.

```graphql
input WalletOrder {
  direction: OrderDirection!
}
```


### Поля

#### `direction` ([`OrderDirection`](/reference/enums/order-direction))

`direction` - указывает направление сортировки.
