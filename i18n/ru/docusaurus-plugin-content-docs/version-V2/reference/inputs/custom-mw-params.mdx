Тип ввода `customMwParams` определяет настраиваемую промежуточную программу для установки.

```graphql
input customMwParams {
  contractAddress: ContractAddressEVM!
  initData: String!
}
```


### Поля

#### `contractAddress` ([`ContractAddressEVM`](/reference/scalars/contract-address-evm))

`contractAddress` - адрес контракта промежуточной программы.


#### `initData` ([`String`](/reference/scalars/string))

`data` - данные инициализации промежуточной программы.
