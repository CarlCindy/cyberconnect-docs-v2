title: SubscribeOrder
---


Тип ввода `SubscribeOrder` определяет порядок, в котором возвращаются результаты.

```graphql
input SubscribeOrder {
  direction: OrderDirection!
}
```


### Поля

#### `direction` ([`OrderDirection`](/reference/enums/order-direction))

`direction` - указывает направление вывода.
