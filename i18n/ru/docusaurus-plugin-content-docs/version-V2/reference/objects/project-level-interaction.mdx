`ProjectLevelInteraction` - это ярлык, описывающий взаимодействие адреса с данным проектом.

```graphql
type ProjectLevelInteraction {
  project: String!
  txCount: Int!
  numSent: Int!
  numReceived: Int!
  firstInteraction: String!
  lastInteraction: String!
  sampleTxHashes: [String!]
}
```
