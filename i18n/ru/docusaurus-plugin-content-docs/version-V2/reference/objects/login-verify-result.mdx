---


Тип `LoginVerifyResult` предоставляет токен доступа для запросов мутации, требующих аутентификации.

```graphql
type LoginVerifyResult {
  accessToken: String!
}
```


### Поля

#### `accessToken` ([`String`](/reference/scalars/string))

`accessToken` - токен доступа для запросов мутации, требующих аутентификации.
