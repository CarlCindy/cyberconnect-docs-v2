Тип `CollectEdge` предоставляет узел `Collect` вместе с его курсором.

```graphql
type CollectEdge {
  node: Collect
  cursor: Cursor!
}
```


### Поля

#### `node` ([`Collect`](/reference/objects/collect))

`node` - узел объекта `Collect`.


#### `cursor` ([`Cursor`](/reference/scalars/cursor))

`cursor` - курсор узла `Collect`.
