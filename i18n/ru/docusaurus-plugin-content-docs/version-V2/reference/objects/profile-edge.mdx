---


Тип `ProfileEdge` предоставляет узел `Profile` вместе с его курсором.

```graphql
type ProfileEdge {
  node: Profile
  cursor: Cursor!
}
```


### Поля

#### `node` ([`Profile`](/reference/objects/profile))

`node` - узел объекта `Profile`.


#### `cursor` ([`Cursor`](/reference/scalars/cursor))

`cursor` - курсор узла `Profile`.
