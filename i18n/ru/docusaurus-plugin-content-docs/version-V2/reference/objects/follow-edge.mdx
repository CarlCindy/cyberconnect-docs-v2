---


Тип `FollowEdge` предоставляет узел `Follow` вместе с его курсором.

```graphql
type FollowEdge {
  node: Follow
  cursor: Cursor!
}
```


### Поля

#### `node` ([`Follow`](/reference/objects/follow))

`node` - узел объекта `Follow`.


#### `cursor` ([`Cursor`](/reference/scalars/cursor))

`cursor` - курсор узла `Follow`.
