Тип `Label` - это общий базовый тип для ярлыков контрактов и/или пользователей. Это могут быть внешние агрегированные или разработанные внутри компании ярлыки.

```graphql
type Label {
  labelName: String!
  labelType: String!
  contractName: String!
  contractAddress: ContractAddressEVM!
  source: String!
}
```


### Поля

#### `labelName` ([`String`](/reference/scalars/string))




#### `labelType` ([`String`](/reference/scalars/string))




#### `contractName` ([`String`](/reference/scalars/string))




#### `contractAddress` ([`ContractAddressEVM`](/reference/scalars/contract-address-evm))




#### `source` ([`String`](/reference/scalars/string))
