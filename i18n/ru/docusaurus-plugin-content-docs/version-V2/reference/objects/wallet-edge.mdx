title: WalletEdge
---


Тип `WalletEdge` предоставляет узел `Wallet` вместе с его курсором.

```graphql
type WalletEdge {
  node: Wallet
  cursor: Cursor!
}
```


### Поля

#### `node` ([`Wallet`](/reference/objects/wallet))

`node` - узел объекта `Wallet`.


#### `cursor` ([`Cursor`](/reference/scalars/cursor))

`cursor` - курсор узла `Wallet`.
