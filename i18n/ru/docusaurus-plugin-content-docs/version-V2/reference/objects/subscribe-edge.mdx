title: SubscribeEdge
---


Тип `SubscribeEdge` предоставляет узел `Subscriber` вместе с его курсором.

```graphql
type SubscribeEdge {
  node: Subscribe
  cursor: Cursor!
}
```


### Поля

#### `node` ([`Subscribe`](/reference/objects/subscribe))

`node` - узел объекта `Subscribe`.


#### `cursor` ([`Cursor`](/reference/scalars/cursor))

`cursor` - курсор узла `Subscribe`.
