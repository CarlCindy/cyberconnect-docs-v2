Тип `ENSInfo` предоставляет информацию о ENS домене.

```graphql
type ENSInfo {
  domain: String!
  avatar: String!
}
```


### Поля

#### `domain` ([`String`](/reference/scalars/string))

`domain` - имя ENS домена.


#### `avatar` ([`String`](/reference/scalars/string))

`avatar` - аватар ENS домена.
