title: EssenceEdge
---


Тип `EssenceEdge` предоставляет узел `Essence` вместе с его курсором.

```graphql
type EssenceEdge {
  node: Essence
  cursor: Cursor!
}
```


### Поля

#### `node` ([`Essence`](/reference/objects/essence))

`node` - узел объекта `Essence`.


#### `cursor` ([`Cursor`](/reference/scalars/cursor))

`cursor` - курсор узла `Essence`.
