---


Тип `FollowStats` предоставляет статистику о последователях пользователя и его последуемых.

```graphql
type FollowStats {
  followerCount: Int!
  followingCount: Int!
}
```


### Поля

#### `followerCount` ([`Int`](/reference/scalars/int))

`followerCount` - общее количество последователей.


#### `followingCount` ([`Int`](/reference/scalars/int))

`followingCount` - общее количество последуемых.
