Тип `GithubInfo` предоставляет информацию о профиле Github.

```graphql
type GithubInfo {
  username: String!
  gistID: String!
  userID: Int!
}
```


### Fields

#### `username` ([`String`](/reference/scalars/string))

`username` - имя пользователя профиля Github.


#### `gistID` ([`String`](/reference/scalars/string))

`gistID` - идентификатор gist, используемый для проверки.


#### `userID` ([`Int`](/reference/scalars/int))

`userID` - идентификатор профиля пользователя Github.
