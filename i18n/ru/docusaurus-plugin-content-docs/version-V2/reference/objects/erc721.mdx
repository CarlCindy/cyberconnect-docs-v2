Тип `ERC721` соответствует стандарту, описанному в https://ethereum.org/en/developers/docs/standards/tokens/erc-721 /.
Добавленные поля включают символ, десятичные дроби.

```graphql
type ERC721 implements Contract {
  contractAddress: ContractAddressEVM!
  name: String!
  contract_creation_time: String!
  block_number: Int!
  block_hash: String!
  bytecode: String!
  symbol: String!
}
```
