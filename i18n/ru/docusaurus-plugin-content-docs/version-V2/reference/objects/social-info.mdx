title: SocialInfo
---


Тип `SocialInfo` предоставляет информацию о социальных профилях идентичности.

```graphql
type SocialInfo {
  twitter: TwitterInfo!
  github: GithubInfo!
}
```


### Поля

#### `twitter` ([`TwitterInfo`](/reference/objects/twitter-info))

`twitter` - профиль в Twitter.


#### `github` ([`GithubInfo`](/reference/objects/github-info))

`github` - профиль на Github.
