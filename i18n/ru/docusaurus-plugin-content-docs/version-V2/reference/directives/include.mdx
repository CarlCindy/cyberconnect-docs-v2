Указывает исполнителю включать это поле или фрагмент только тогда, когда аргумент `if` имеет значение true.

```graphql
directive @include(
  if: Boolean!
)
```


### Аргументы

#### `if` ([`Boolean`](/reference/scalars/boolean))

Включается, когда значение true.
