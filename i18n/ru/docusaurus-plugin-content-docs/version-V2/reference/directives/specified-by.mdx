Предоставляет URL-адрес, который определяет поведение этого скаляра.

```graphql
directive @specifiedBy(
  url: String!
)
```


### Аргументы

#### `url` ([`String`](/reference/scalars/string))

URL-адрес, который определяет поведение этого скаляра.
