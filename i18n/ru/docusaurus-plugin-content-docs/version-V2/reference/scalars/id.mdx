title: ID
---


Скалярный тип `ID` представляет собой уникальный идентификатор, часто используемый для повторной выборки объекта или в качестве ключа для кэша. Тип ID отображается в ответе JSON в виде строки; однако он не предназначен для чтения человеком. Когда в качестве типа ввода ожидается любая строка (например, `"4"`) или число (например, `4`) входное значение будет принято в качестве ID.

```graphql
scalar ID
```
